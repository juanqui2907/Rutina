# Modo Mejor — PWA minimalista

App PWA para celular con:

- Misiones grandes divididas en pasos pequeños.
- Checklist diario de 5 hábitos.
- Alcancía y contador de 10 fallas por mes.
- Gráfica mensual de fallas.
- Horario semanal replicable y editable.
- Cuaderno mental por temas: TDAH, ansiedad, depresión, sobrepensar, autoestima, etc.
- Exportación/importación de datos.
- Modo claro/oscuro.
